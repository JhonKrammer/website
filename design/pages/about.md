## 1. 页面目标

讲好 Dark Matter 的故事——为什么存在、叫这个名字的意义、相信什么。文字为主，设计为辅，让内容本身成为主角。

## 4. 各区块详细规格

### 4.1 Page Header

- **上下 padding：** `80px` (桌面), `48px` (移动)
- **背景：** `#0A0B10`，底部有 radial-gradient 微光
- **标题：** `H1, 48px (桌面) / 32px (移动), #F0F0F5, font-weight 700, 居中`
- **渐变线：** `60px × 3px`, Primary Gradient, 居中, 标题下方 `16px`
- **副标题：** `Body L, 18px (桌面) / 16px (移动), #9CA3B0, 居中`, 线下方 `16px`

### 4.2 Story Section（我们的故事）

- **上下 padding：** `96px` (桌面), `48px` (移动)
- **背景：** `#0A0B10`
- **内容最大宽度：** `720px`, `margin: 0 auto`
- **Section Title：** `H2, 36px (桌面) / 28px (移动), #F0F0F5, 居中`
- **渐变线：** 标题下方 `12px`, `40px × 3px`
- **正文：**
  - `font-size: 16px`, `color: #F0F0F5`, `line-height: 1.8`
  - 段落间距: `margin-bottom: 24px`
  - 段首不缩进（现代 web 风格）
  - 段间可插入 `margin-bottom: 40px` 作为章节分隔

### 4.3 Dark Matter 含义（引言区块）

- **上下 padding：** `96px` (桌面), `48px` (移动)
- **背景：** `#12141D` (交替背景)
- **引言容器：**
  - 最大宽度 `640px`, 居中
  - 左边框 `3px solid #6C8EEF`
  - 内边距 `0 0 0 32px`
- **引号文字：**
  - `font-size: 24px (桌面) / 20px (移动)`
  - `color: #F0F0F5`
  - `font-weight: 500`
  - `line-height: 1.6`
  - `font-style: italic`
- **作者行：** 引言下方 `16px`, `font-size: 14px`, `color: #9CA3B0`
- **补充段落：** 引言区块下方 `40px`, `max-width: 720px`, `margin: 0 auto`, Body 样式

### 4.4 Vision & Mission

- **上下 padding：** `96px` (桌面), `48px` (移动)
- **背景：** `#0A0B10`
- **2 列网格：** `grid-template-columns: repeat(2, 1fr)`, `gap: 24px`
- **每列：**
  - 背景 `#12141D`
  - 圆角 `16px`
  - 内边距 `32px`
  - 标题 `H3, 24px, #F0F0F5`
  - 渐变线 `40px × 3px`, 标题下方 `12px`
  - 核心语句 `Body L, 18px, #F0F0F5, font-weight 500`, 线下方 `16px`
  - 扩展描述 `Body S, 14px, #9CA3B0, line-height 1.6`
- **移动端：** 单列堆叠

### 4.5 Beliefs Section（我们的信念）

- **上下 padding：** `96px` (桌面), `48px` (移动)
- **背景：** `#12141D` (交替)
- **Section Title：** 居中
- **信念列表容器：** `max-width: 720px`, `margin: 0 auto`
- **每条信念：**
  - `display: flex`, `gap: 24px`
  - `padding: 24px 0`
  - 底部边框 `1px solid rgba(255,255,255,0.06)` (最后一条无边框)
- **序号：** `width: 48px`, `flex-shrink: 0`
  - `font-size: 14px`, `color: #A78BFA`, `font-weight: 600`
  - 格式: `01`, `02`...
- **标题：** `H3, 20px, #F0F0F5, font-weight 600`
- **描述：** `Body S, 14px, #9CA3B0, line-height 1.6`, 标题下方 `4px`

## 5. 交互说明

| 交互 | 触发 | 效果 |
|------|------|------|
| 页面加载 | DOM ready | Page Header fadeInUp, 0.6s |
| 滚动到故事 | 进入视口 | 正文段落依次 fadeInUp, stagger 150ms |
| 滚动到引言 | 进入视口 | 引言区块 fadeIn, 左边框从下到上展开 (CSS height 动画) |
| 滚动到愿景使命 | 进入视口 | 2 张卡片 fadeInUp, stagger 100ms |
| 滚动到信念 | 进入视口 | 信念条目依次 fadeIn, stagger 80ms |
| 愿景/使命卡片 hover | 鼠标悬停 | `translateY(-4px)`, `border-color` 变亮 |
| 信念条目 hover | 鼠标悬停 | 背景微亮 `rgba(108,142,239,0.03)`, `transition: 0.2s` |

---

## 6. 响应式方案

| 断点 | 变化 |
|------|------|
| `< 768px` | 标题 32px; 内容 px-5; 愿景使命单列; 信念序号缩为 32px 宽 |
| `768-1023px` | 内容 px-8; 愿景使命保持 2 列; 标题 40px |
| `≥ 1024px` | 完整布局; 内容区 max-width 720px / 1200px; 所有间距满值 |
