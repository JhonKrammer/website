## 1. 项目背景

Dark Matter 是一家由 Leon 创立的新型公司，核心团队由 1 位人类 CEO + 8 位 AI Agent 组成。公司官网是对外展示公司形象和团队构成的第一窗口。

## 2. 项目目标

- 建立 Dark Matter 品牌的线上存在感
- 清晰传达「一人公司 + AI Agent 团队」的创新模式
- 展示每位团队成员的角色与能力

## 3. 目标用户

- 潜在合作伙伴 / 客户
- 对 AI Agent 协作模式感兴趣的人
- 投资人 / 行业观察者

## 4. 范围

### v1.0 包含（本次开发）
| 页面 | 说明 |
|------|------|
| 首页（Home） | 公司概览、核心价值主张、团队预览 |
| 关于我们（About） | 公司故事、愿景使命、Dark Matter 的含义 |
| 团队（Team） | 9 位成员的详细介绍卡片 |

### 后续版本（暂不开发）
- 产品/项目展示
- 博客
- 联系方式
- 招贤纳士

## 5. 设计原则

- **风格：** 暗色调 + 宇宙/深空主题（呼应 "Dark Matter" 品牌名）
- **调性：** 专业但不冰冷，创新但不浮夸
- **响应式：** 移动端优先，桌面端完整体验
- **性能：** 静态生成，加载速度 < 2s

## 6. 技术方案

| 维度 | 选型 | 理由 |
|------|------|------|
| 框架 | **Next.js 14+ (App Router)** | SSG 支持、React 生态、图片优化、部署方便 |
| 样式 | **Tailwind CSS** | 快速开发、一致性强、暗色模式原生支持 |
| 动效 | **Framer Motion** | 轻量级动画，适合页面过渡和卡片交互 |
| 部署 | **Vercel** | 零配置部署、自动 CI/CD、免费额度够用 |
| 域名 | 自定义域名（待定） | 如 `darkmatter.ai` 或类似 |

## 7. 成功指标

- [ ] 网站可正常访问，移动端和桌面端均可浏览
- [ ] 页面加载时间 < 2s
- [ ] 公司介绍信息准确完整
- [ ] 9 位团队成员信息全部展示
